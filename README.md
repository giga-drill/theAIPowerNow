# AI 能力普及课前调研问卷

这是一个纯静态问卷页面，用于收集制造业员工在 AI 普及课程前的岗位背景、工作痛点和 AI 接触程度。

## 文件

- `index.html`：问卷首页，已配置 Netlify Forms。
- `thank-you.html`：提交成功页。
- `styles.css`：页面样式。
- `script.js`：填写进度提示。
- `netlify.toml`：Netlify 静态部署配置。

## 推荐部署方式：Netlify

1. 把这个目录上传到 GitHub 仓库。
2. 在 Netlify 新建站点，选择该仓库。
3. 构建命令留空，发布目录填 `.`。
4. 部署后进入 Netlify 的 `Forms` 页面，确认 `ai-readiness-survey` 表单被识别。
5. 收到答卷后可在表单详情页导出 CSV。

Netlify Forms 只会在部署后的线上站点处理提交；本地直接打开 HTML 可以预览页面，但不会保存提交结果。

## 保密提醒

问卷首页已经提示不要填写涉密或敏感信息。正式发放时，建议在通知里再次说明：

> 请不要填写具体型号、图纸参数、工艺参数、客户/供应商敏感信息或内部未公开数据，只描述工作类型和痛点。

## 可调整内容

如果要让问卷更短，优先保留这些字段：

- 岗位类型
- 希望 AI 帮忙的真实场景
- AI 接触程度
